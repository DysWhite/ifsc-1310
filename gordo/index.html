My software and what it's used for

Blender
Open source 3D software. It can handle everything from modelling to the final render, and everything in between. It even has a built in ray tracing rendering engine called Cycles.
FL Studio
Paid Digital Audio Workstation, or DAW. While FL doesn't come with many synths itself, it makes it easy to make individual parts of a song, and also to string those parts together into one cohesive track.
Visual Studio
I'm sure most reading this already know what VS is, but for those that don't, it's an Integrated Developer Environment, or IDE. It makes writing code a lot easier, and provides helpful tools for debugging.

I use other software and stuff, but it's common things like Word, and stuff like that. So, not super worth talking about.
